What is Tandem Browser?

Tandem Browser is a browser built specifically for collaboration with OpenClaw. While you browse normally, Wingman can watch along, help, and perform actions through Tandem's local browser API while staying outside the page context.

Tandem is maintained in the same ecosystem as OpenClaw and should be understood as a first-party companion browser, not a gimmick integration or a generic browser shell with chat bolted on later.

Wingman Panel — Your AI Assistant

The Wingman panel (Cmd+K) is where your AI assistant lives. It has multiple tabs:

Chat

Chat directly with Wingman via text or voice. It can help with browsing, answer questions, and perform actions.

Activity Log

Wingman follows all your browsing activity in real-time: which pages you visit, what you click, what you type. This helps it understand context.

Screenshots

All screenshots you take are visible here with thumbnails. Click to view the full image.

Memory

Wingman remembers important information about websites you visit — perfect for finding information later.

💡 Voice Tip Hold Cmd+M to talk. Release and Wingman processes your speech automatically!

Drawing & Annotation

Press Cmd+Shift+D to toggle drawing mode. You can then draw directly on web pages.

Available Tools

✏️ Free Line

Draw freehand shapes, circles, underlines.

🔸 Rectangles

Highlight sections and blocks with rectangles.

⭕ Circles

Highlight specific elements with circles.

📝 Text Labels

Add text notes to your annotations.

Choose from different colors: red (default), yellow, green, and blue. Press "📸 Snap for Wingman" to take a screenshot with all your annotations.

Voice Input

Tandem has built-in speech recognition for Dutch and English input.

How to use

  1. Press Cmd+M to activate voice input
  2. You'll see a red pulsing indicator — start talking
  3. Wingman transcribes your speech live in the chat panel
  4. After a brief silence the message is sent automatically

Voice input works perfectly with annotations — draw something and explain it via voice!

Tabs & Groups

Tandem supports advanced tab management with colors and groups.

Tab Navigation

Use Cmd+1 through Cmd+9 to quickly switch between tabs. Cmd+T opens a new tab, Cmd+W closes the current tab.

Tab Indicators

Tabs show different indicators:

  • 👤 — Tab controlled by you
  • 🤖 — Tab controlled by Wingman (AI)
  • 🎵 — Tab is playing audio
  • 🔴 — Audio is being recorded

Memory & Data

Tandem remembers everything you do to help you better later.

Site Memory

For every website Tandem stores: title, content, forms, links, and changes over time. Search through your browse history via the Memory tab in the Wingman panel.

Form Memory

Tandem remembers which forms you fill in and can auto-fill them later. Sensitive data like passwords is stored encrypted.

Behavioral Learning

Tandem learns your browsing behavior (timing, mouse movements, typing patterns) to mimic more human-like behavior during automated actions.

Stealth & Privacy

Tandem is designed to be completely invisible to websites. It looks exactly like a normal Chrome browser.

Anti-Detection Features

  • Canvas fingerprint randomization
  • WebGL renderer/vendor spoofing
  • Audio context fingerprint spoofing
  • Timing protection
  • Font enumeration masking
  • Humanized delays and behavior simulation
🛡️ Privacy All data stays local on your computer. Tandem NEVER sends data to external servers.

Tips & Tricks

Form Memory

Advanced Features

Tandem has many advanced features not visible in the UI:

  • Headless Browsing: Wingman can invisibly visit websites
  • Scheduled Watches: Automatically monitor websites for changes
  • Network Inspector: API discovery and request monitoring
  • Extension Support: Load Chrome extensions
  • Audio Capture: Record tab audio for later
